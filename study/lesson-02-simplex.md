# 第 2 课:单纯形法 —— LP 的通用算法

承接 [[lesson-01-linear-programming]]。图解法只能处理两变量;单纯形法是任意维度的通用算法。

## 1. 核心思想

利用顶点定理(最优解在顶点上):

> 🎯 从一个顶点出发,每次沿一条边「爬」到**目标更优的相邻顶点**,直到所有邻居都不更优——即最优。

像爬山:往更高方向迈步,迈不动了就到顶。

## 2. 松弛变量:把 ≤ 变 =

算法只处理等式。每条 `≤` 约束加一个**松弛变量(slack)**,表示「这项资源还剩多少」:

```
x +  y + s1 = 10   (s1 = 剩余土地)
x + 2y + s2 = 16   (s2 = 剩余工时)
s1, s2 ≥ 0
```

## 3. 跟着农场题跑一遍

- **起点 (0,0)**:x=y=0 → s1=10, s2=16(资源全剩),z=0。
- **第一跳**:目标 `z=40x+50y`,y 系数 50 最大 → y 进基。
  比值检验:土地 `10/1=10`,工时 `16/2=8` → 取小者 8,工时先用光。
  到 **(0,8),z=400**。
- **第二跳**:重写目标得 `z = 400 + 15x - 25s2`,x 系数 +15>0 → x 还能涨。
  比值检验 → x 最多到 4,土地用光。到 **(4,6),z=460**。
- **停**:再重写目标得 `z = 460 - 30s1 - 10s2`,系数全负 → 再涨只会更小 → 最优。

```
(0,0) ──→ (0,8) ──→ (4,6)    沿边爬山,没碰 (10,0)
 z=0      z=400    z=460
```

✅ 最优 x=4, y=6, z=460,与图解法一致。变量越多,「不用全枚举」的优势越大。

## 4. 记住四个关键词就懂单纯形

| 术语 | 作用 | 直觉 |
|------|------|------|
| **松弛变量 slack** | ≤ 变 = | 资源还剩多少 |
| **进基变量** | 选目标改善最快的 | 往最陡方向爬 |
| **比值检验** | 防资源变负 | 哪个资源先用光就停那 |
| **最优判据** | 目标行系数全 ≤0 | 四周没有更高处 |
