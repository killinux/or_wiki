# 第 3 课:对偶与影子价格 —— 从「算出解」到「读懂解」

承接 [[lesson-02-simplex]]。对偶是 LP 最精彩的部分:它回答「**多给 1 单位资源,利润能多赚多少**」。

## 1. 换个视角看农场题

原问题(**原始问题 primal**)站在**农场主**角度:怎么种地,利润最大?
```
max  40x + 50y
s.t. x +  y ≤ 10   (土地)
     x + 2y ≤ 16   (工时)
     x, y ≥ 0
```

现在换个人:一个**收购商**想把你的土地和工时全买走。他得给每种资源**定个价**:
- `u1` = 每单位土地的价
- `u2` = 每单位工时的价

收购商想**少花钱**(min),但出价必须高到让你愿意卖——也就是说,买走「造 1 公顷小麦所需的资源」付的钱,不能少于你自己种小麦能赚的 40:

## 2. 对偶问题(dual)

```
min  10 u1 + 16 u2          (总收购支出最小)
s.t. u1 +  u2 ≥ 40          (小麦那一列:种小麦的资源不能贱卖)
     u1 + 2u2 ≥ 50          (玉米那一列)
     u1, u2 ≥ 0
```

**对偶的机械构造规律**(原始 max、约束 ≤ → 对偶 min、约束 ≥):
- 原始**约束**个数 = 对偶**变量**个数(2 条约束 → u1,u2)
- 原始**变量**个数 = 对偶**约束**个数(x,y → 2 条对偶约束)
- 原始约束的**右端值**(10,16)→ 跑到对偶**目标**里当系数
- 原始目标系数(40,50)→ 跑到对偶约束的**右端**
- 系数矩阵**转置**(行列对调)

## 3. 强对偶定理

> 🎯 **原始问题的最优值 = 对偶问题的最优值。** 两个视角殊途同归。

解对偶:最优时 x>0、y>0,对应两条对偶约束取等(**互补松弛**):
```
u1 +  u2 = 40
u1 + 2u2 = 50   →  相减 u2 = 10, 代回 u1 = 30
```
对偶目标 `10·30 + 16·10 = 460` = 原始最优 460 ✅ 强对偶成立。

## 4. 影子价格(shadow price)—— 对偶变量的真正含义

`u1=30, u2=10` 不只是「价」,它们就是**影子价格**:

> 🎯 **某资源的影子价格 = 该资源每增加 1 单位,最优目标值的增加量。**

- 土地影子价格 = **30**:多 1 公顷地 → 利润 +30。
- 工时影子价格 = **10**:多 1 工时 → 利润 +10。

**验证土地**:把土地从 10 改成 11 重解 → x=6,y=5,利润 490 = 460+30 ✅
**验证工时**:把工时从 16 改成 17 重解 → x=3,y=7,利润 470 = 460+10 ✅

## 5. 影子价格的两条实战铁律

1. **它告诉你该花钱买哪种资源**:土地(30)比工时(10)金贵。若买 1 单位地花费 < 30,就该买;
   买 1 单位工时花费 < 10 才值。这是**资源投资决策**的依据。
2. **它只在一个范围内有效(灵敏度分析)**:土地影子价格 30 仅在土地 ∈ [8,16] 内成立
   (超出后会有别的约束变紧,价格跳变)。所以「多赚 30」不能无限外推。

> 若某约束在最优时**没用满**(有剩余/松弛),它的影子价格 = 0 —— 这种资源再多也不涨利润。

## 6. 一句话总结

- **原始**:做什么决策。**对偶**:每种资源值多少。二者最优值相等(强对偶)。
- **影子价格 = 对偶变量 = 资源的边际价值**,是 OR 里连接「数学解」与「商业决策」的桥。
