# 第 1 课:线性规划 LP

承接 [[_roadmap]]。线性规划是 OR 的地基。

## 1. 什么是线性规划

在一堆**线性约束**下,求一个**线性目标**的最大或最小值。"线性"=变量都只有一次方,
没有 `x²`、`xy`、`√x`。

## 2. 建模五件套(OR 的核心动作)

以「工厂排产」为例:造桌子赚 30/张、椅子赚 20/把;木料共 100(桌 5、椅 2),工时共 48(桌 3、椅 1)。

| 术语 | 在例子里 | 解释 |
|------|----------|------|
| **决策变量 decision variable** | `x`=桌子数, `y`=椅子数 | 你能直接做主、求解要求的量 |
| **目标函数 objective** | `max 30x + 20y` | 要最大化/最小化的线性式(max/min) |
| **约束 constraint** | `5x+2y≤100`、`3x+y≤48` | 不能突破的线性限制,`≤ ≥ =` 是方向 |
| **LP** | 上述这种,变量可取小数 | Linear Programming,OR 第一课 |
| **求解器 solver** | Gurobi / COPT / PuLP | 自动算最优解的软件 |

> 🎯 **心法 1:决策变量要选「你能直接拍板的量」,并让目标和约束的系数最直白。**
> 反例:用「工时」而非「公顷」当变量,会导致系数要换算、容易算错。

## 3. 农场练习题(标准答案)

10 公顷地种小麦(40/公顷, 1 工时)或玉米(50/公顷, 2 工时),共 16 工时,利润最大。

令 `x`=小麦公顷, `y`=玉米公顷:

```
max  40x + 50y
s.t. x +  y ≤ 10    (土地)
     x + 2y ≤ 16    (工时)
     x, y ≥ 0       (非负)
```

非负约束几乎每题都有,别忘。

## 4. 图解法(两变量专用,建立直觉)

1. **画约束**:把每个不等式当等式,找两个截距连成边界线;`≤` 取靠原点一侧。
   - `x+y=10` → 过 (10,0)、(0,10)
   - `x+2y=16` → 过 (16,0)、(0,8)
2. **可行域**:所有约束的公共区域。本题是四边形,顶点 `(0,0)、(10,0)、(4,6)、(0,8)`。
   - 交点 (4,6):联立两线相减得 `y=6`,代回 `x=4`。
3. **顶点定理(LP 最重要的结论)**:

   > 🎯 **LP 的最优解一定在可行域的某个顶点上。** 于是只需枚举有限个顶点。

4. **枚举顶点取最优**:

   | 顶点 | 含义 | 利润 40x+50y |
   |------|------|--------------|
   | (0,0) | 啥也不种 | 0 |
   | (10,0) | 全小麦 | 400 |
   | (0,8) | 全玉米 | 400 |
   | **(4,6)** | **4 小麦 + 6 玉米** | **460** ✅ |

## ✅ 结论

**4 公顷小麦 + 6 公顷玉米,最大利润 460。** 此时两约束都「用满」(土地、工时都到上限),
最优解常正好卡在多约束的交点。
