# AEL：用大语言模型做算法演化 (Algorithm Evolution Using Large Language Model)

> 一句话：在演化计算框架内用 LLM 自动生成并变异**整个优化算法**(算法级演化、无需训练),把 "LLM + 演化" 确立为自动算法发现的范式,是 Evolution of Heuristics (EoH) 的概念前身。

## 关键内容
- **算法级演化**:LLM 充当生成与变异算子,在演化循环中产出并改写完整算法(individual = 一段算法/启发式),全程无任何模型训练或微调。
- 在 **TSP(旅行商问题)的构造式方法(constructive method)** 上做演示:用 LLM 演化"下一步选哪个城市"的构造规则。
- 演化出的构造算法**优于简单手工启发式**(如贪心/最近邻)**也优于直接让 LLM 一次生成的 naive 启发式**;并在**不同 TSP 规模**上表现出可扩展性(可泛化到演化时未见过的问题规模)。
- 与 **FunSearch(2023 年末)并行**,共同确立了 "LLM + 演化计算" 自动发现算法/启发式这一方向。
- 重叠作者(Fei Liu, Qingfu Zhang 等)直接通向后续的 **Evolution of Heuristics (EoH)**,AEL 可视为其雏形。
